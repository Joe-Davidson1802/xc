## Task Scripts

A script is what defines a tasks behaviour.

## Syntax

To define a script you must add a code block under the name of a task.

````markdown
## Tasks
### Task1
```
echo "Hello 世界!"
echo "Hello العالمية!"
echo "Hello ертөнц!"
```
````
